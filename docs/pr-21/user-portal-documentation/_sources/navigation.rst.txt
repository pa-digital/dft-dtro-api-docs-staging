Navigation
==========

Once logged in, you will see the main navigation screen. The view you see here depends on whether you have a publisher or consumer account, and which environment you have logged in to (integration or production).

Integration environment
***********************

In the integration environment, the following options are available from the navigation screen.

* Create a new test publisher application (publisher accounts only)
* Create a new test consumer application
* Request access to the production environment
* View apps

Production environment
**********************

In the production environment, the following options are available from the navigation screen.

* Create a new production publisher application (publisher accounts only)
* Create a new production consumer application
* View apps
* Submit an error report
